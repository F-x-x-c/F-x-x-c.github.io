1. 字符函数

concat: 表示字符串的连接 等同于||
select concat(‘my name is ‘, ename) from emp;

2. 首字符大写函数

将字符串的首字符大写
select initcap(ename) from emp;

3. 全部字符大写函数

将字符串的全部字母大写
select upper(ename) from emp;

4. 全字符小写函数

select lower(ename) from emp;

5. 填充字符串函数

使用 * 去向左补全ename
select lpad(ename, 10, ‘*’) from emp;

使用 * 向右补全ename
select rpad(ename, 10, ‘*’) from emp;

6. 去除空格

去除字符串左右两边的空格, 但不包括字符串中间的;
select trim(ename) from emp;

去除字符串左边的空格;
select ltrim(ename) from emp;

去除字符串右边的空格;
select rtrim(ename) from emp;

7. indexOf函数

查询字符在字符串中第一次出现的位置
select instr(ename, ‘A’) from emp;

8. 查看字符串长度

select length(ename) from emp;

9. 替换

select replace(ename, ‘ai’, ‘hehe’) from emp;

2. 数值函数

2.1 四舍五入

select round(123.1235, 3) from dual;
select round(123.123, 2) from dual;

2.2 截断数据

截断数据但是不进行四舍五入
select trunc(123.127, 2) from dual;

2.3 取模操作

select mod(10, 4) from dual;

2.4 向上\下取整

select ceil(12.12) from dual;
select floor(13.99) from dual;

3. 条件函数

3.1 decode

decode(条件,值1,翻译值1,值2,翻译值2,…值n,翻译值n,缺省值)　　
　　该函数的含义如下：　　
　　IF 条件=值1 THEN
　　RETURN(翻译值1)
　　ELSIF 条件=值2 THEN
　　RETURN(翻译值2)
　　……
　　ELSIF 条件=值n THEN
　　RETURN(翻译值n)　　
　　ELSE
　　RETURN(缺省值)
　　END IF

如果说”条件”(也就是第一个参数)等于”值1”, 那么就返回”翻译值1”, 如果”条件”等于”值2”, 那么就返回”翻译值2”, 如果都不满足, 那么就返回”缺省值”, 类似于if-then-else if-then-else-then的各个编程语言中的函数;

3.2 case when

case 查询的值
when x1 then
y1
when x2 then
y1
when x3 then
y3
……
when xn then
yn
else
yn+1
end

其实就是经典编程函数中的switch-case-break语句;

5. 子句顺序

字句 说明 是否必须使用
select 要返回的列或表达式 是
from 从中检索数据的表 仅在从表选择数据时使用
where 行级过滤 否
group by 分组说明 仅在按组计算聚集时使用
having 组级过滤 否
order by 输出排序顺序 否

首先需要确定查找的是哪一张表, 也就是from, 代表从哪一张表中拿数据; 然后需要对这张表的数据进行过滤, 也就是where, 行级过滤; 再按照group by语句规定显示分组的依据, 同时提取满足having子句中组条件表达式的那些组; 再再通过select给出的选择列表对数据进行选择性显示; 然后通过order by提供的顺序对数据进行排序, 最后显示数据;

组函数的定义是对某个分组求某个值, 如果不指定group by的话就是对表中的所有数据求值, 比如求所有部门的平均薪资就不需要指定group by, 而如果要求每个部门各自的平均薪资, 那就需要用group by指定deptno, 指定之后, 就会选择相同的部门编号的人的薪资进行求平均;

看上去是一种层层筛选, 首先选表, 然后选行, 然后分组(通过分组依据进行数据的操作), 再选组, 再者选择要显示哪些数据, 再然后排序, 最后显示;

  • 题目: 求部门下雇员的工资大于2000的人数;

select deptno, count(1) from emp where sal > 2000 group by deptno;

  • 题目: 求部门薪水最高;

select deptno, max(sal), job from emp group by deptno, job;

如果需要同时查询名字和编号的话, 就不想上面那么简单了, 因为deptno是出现在group by的选择列表中的, 所以可以显示在select的选择列表中, 而ename并没有, 所以需要像下一道题这样写;

  • 题目: 求部门中工龄最小和工龄最大的人的信息;

    select *
    from emp
    where hiredate in (select min(hiredate) from emp group by deptno)

         or hiredate in (select max(hiredate) from emp group by deptno)

6. 关联查询

想要同时查询分别在两张表中的数据时需要使用关联查询;

关联查询涉及一条语句两张表的查询, 这两张表需要通过where语句进行关联, 关联的方式有4种;

  1. 等值连接

    对两张表所拥有的共同的数据进行查询, 比如使用deptno对emp和dept进行关联查询, 所得到的数据就为两张表的数据:
    select * from emp, dept where emp.deptno = dept.deptno;

  2. 非等值连接
    <,>,<=,>=,!=连接时称非等值连接; 一张表的某项数据是另外一张表的某项数据的子集, 比如查询雇员名称, 以及自己的薪水等级, 这两个数据分别在两张表中;

    select e.ename, e.sal, sg.grade
    from emp e, salgrade sg
    where e.sal between sg.losal and sg.hisal;
    也就是说e.sal在[sg.losal, sg.hisal]这个区间上的值全部给我显示出来;

  3. 外连接
    分为左外连接和右外连接;
    左外连接: 表示把左表的数据全部显示, 不管数据有没有关联到右表;
    select * from emp e, dept d where e.deptno = d.deptno(+);

    右外连接类似;
    select * from emp e, dept d where e.deptno(+) = d.deptno;

    注意: 不可以在等好的左右两边同时添加(+);

  4. 自连接
    也就是把一张表当成两张表来查;

    • 将雇员和他的经理查出来;

      select e.ename, m.ename boss from emp e, emp m where e.mgr = m.empno;

2. 出现的问题

3. 操作

对用户解锁/上锁 alter user scott acoount unlock/lock;

登录scott: 打开命令行, 输入 sqlplus /nolog; 输入 conn scott/tiger@orcl;

查看表 select * from tab;

设置数据表显示的大小 set pagesize 50; set linesize 200;

查看数据 select * from emp;

查询表的数据需要指定用户, 即使是管理员也不可以随意查询表数据, 比如管理员账户查询数据可用下面这条语句 select * from scott.emp;

给表添加注释(Comment) comment on table emp is ‘雇员表’;

给列添加注释(Column) comment on column emp.ename is ‘雇员’;
